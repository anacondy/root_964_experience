## Quick Start

### Option 1: Use the Live Site (Recommended)

Simply visit [https://anacondy.github.io/root_964_experience/](https://anacondy.github.io/root_964_experience/) in any modern web browser.

### Option 2: Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/anacondy/root_964_experience.git
   cd root_964_experience
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Navigation

### Home Page

The home page displays the main poster artwork. Click on the central image to enter the dashboard.

### Dashboard

The dashboard contains four main modules:

- **Overview** - View all uploaded files in a grid layout with filtering and sorting
- **Modules** - Reserved for future expansion
- **Database** - Network speed testing and diagnostics
- **Upload** - File upload and code execution area

## First Steps

1. **Enter the System** - Click the poster image on the home page
2. **Upload Files** - Use the Upload module to add files (images, videos, code, etc.)
3. **View Files** - Navigate through uploaded files using arrow keys or clicking
4. **Test Network** - Switch to Database tab and run a speed test

## Tips

- Enable sound for the full immersive experience
- Use a modern browser for best performance
- Enable hardware acceleration in your browser settings
- For code execution, use the sandbox mode for safety
